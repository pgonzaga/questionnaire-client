# Questionnaire Client

### Getting started

 ```bash
 git clone git@github.com:pgonzaga/questionnaire-client.git
 cd questionnaire-client
 npm install
 npm start
  ```
  
  ### Running the test suite
  
   ```bash
 npm test
  ```
